# Product Category Classification
A machine learning project for predicting product categories based on product titles.
This project is aiming to automatically classify products into predefined categories.

## Technologies
- Python 3.13
- pandas
- scikit-learn
- TF-IDF
- LinearSVC

## How to Run

1. Clone the repository
   git clone https://github.com/Sanch004/predict-category-of-product.git
2. Install dependencies
   pip install -r requirements.txt
3. Train the model
   training_model.py
4. Test the model
   testing.py

## Example

Input:
iphone 7 32gb gold Apple iPhone

Output:
Mobile Phones
